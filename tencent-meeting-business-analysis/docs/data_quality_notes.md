# Data Quality Notes

## 1. 腾讯会议收入缺失

腾讯会议不是独立披露主体，腾讯财报未单独披露腾讯会议收入。因此，本项目不对腾讯会议收入做确定性估算，仅使用收入同比增速、产品价格体系和腾讯ToB板块数据作为辅助判断。

## 2. 用户口径限制

腾讯会议公开披露的主要是累计用户数，不能等同于MAU或DAU。DAU仅在2020年初披露过一次，MAU公开口径存在冲突，因此不作为核心趋势指标。

## 3. 腾讯板块数据限制

腾讯金融科技及企业服务收入包括支付、理财、云服务、企业服务等多类业务，不能代表腾讯会议收入。本项目仅将其作为腾讯ToB业务背景指标。

## 4. 行业市场口径限制

IDC、艾瑞、艾媒、CNNIC等机构的统计口径不同。除非来源机构和统计口径一致，否则不将不同机构数据拼接成同一趋势线。

## 5. 竞品口径限制

Zoom使用财年口径，钉钉和飞书多为发布会或媒体披露的关键节点数据。不同竞品之间的用户数、组织数、付费企业数不可直接等价比较。

## 6. 数据使用建议

正式报告中优先使用可信度为“高”或“中高”的数据。标注“推算”“预计”“第三方估算”的数据应放在辅助说明中，不作为核心结论依据。
